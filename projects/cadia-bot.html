Cadia Bot

Advanced multipurpose Discord bot with WIP RPG-style progression

Overview

Cadia Bot is a feature-rich Discord bot combining powerful moderation tools, utility commands, and an WIP RPG system. Users gain experience by interacting, complete quests, battle enemies, and build their character, turning regular server activity into an immersive game.

Key Features

  • WIP RPG system
  • Economy with currency, shops, gambling, and trading
  • Advanced moderation and auto-moderation tools
  • Custom server-specific commands and configurations
  • Daily quests and event systems

Tech Stack

JavaScript Node.js Discord.js Sapphire Framework MongoDB
