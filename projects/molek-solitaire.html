Molek-Syntez Solitaire Solver

AI-powered solver for complex chemical card puzzles

Overview

This project implements an AI solver for the challenging solitaire minigame in Zachtronics' Molek-Syntez, where cards must be stacked following strict chemical bonding rules. Many levels are nearly impossible to solve manually.

Key Features

  • Breadth-first search with constraint satisfaction
  • Explores millions of move combinations efficiently
  • Finds optimal or valid solutions in seconds
  • Outputs step-by-step move sequences
  • Handles complex rule sets accurately

Tech Stack
